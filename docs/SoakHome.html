Soak

A UX e-commerce project focused on creating a seamless digital experience for a fictional bath bomb brand. From branding to interactive product flows, my team designed a prototype that prioritized both user delight and usability. This team project was done at Indiana University, during my senior year, in the course Visual Design For the Web.

Project Overview

Soak is a modern bath bomb e-commerce concept designed to capture a relaxing yet playful brand identity. Through user research, branding exploration, and interface design, my team built a high-fidelity shopping experience that simplifies product discovery and checkout.

My Role

  • Led user flow mapping for the shopping and checkout experience
  • Contributed to visual branding, logo design, and packaging concepts
  • Built high-fidelity prototypes and led usability testing sessions

Key Features

  • Interactive product cards with hover animations and scent descriptions
  • Streamlined checkout experience with limited user input steps
  • Consistent, mobile-friendly layout with visual brand storytelling
