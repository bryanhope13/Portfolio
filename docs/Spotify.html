Spotify Social Features Redesign

Spotify is one of the most popular music streaming platforms in the world, but its social sharing features remain underdeveloped compared to its personalized mixes and curated experiences. This redesign project explores how adding a native social layer can make music sharing more personal, integrated, and dynamic for users.

Project Overview

The current Spotify mobile experience forces users to share songs via third-party apps, resulting in scattered, impersonal exchanges. The desktop version includes a small friend activity feed, but mobile users lack an equivalent. This redesign introduces a Social Sharing Hub that brings social connection and real-time interaction directly into the appâ€”without relying on external platforms.

  • Refined interface through iterative user feedback and testing

Key Features

  • Integrated Notifications: Real-time feedback on shared activity

Methods Used

  • User Journey Mapping: Analyzed existing flow for sharing songs and identified friction points.
  • Low-Fidelity Prototyping: Sketched new flows for Social Hub integration.
  • Interactive Wireframes: Designed mobile interface screens showing posts, comments, and notifications.
  • Comparative Analysis: Examined social features in other apps to inform interaction patterns.
  • Usability Testing: Gathered feedback on discoverability and ease of sharing.
