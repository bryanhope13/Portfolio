Data Dignity

A mobile privacy and data management concept that helps users understand where their data lives, who can access it, and what actions to take—designed for quick scanning, reduced clutter, and calm decision-making.

Project Overview

Data Dignity explores a growing problem: personal data is collected constantly and stored across many systems—school tools, apps, devices, and third-party services. This concept gives users a single hub to see their data at a glance and take meaningful steps to reduce risk. The experience prioritizes clarity and focus, using a “snapshot” dashboard that summarizes storage locations, permissions, and recent changes without overwhelming the user.

Target User

The interface is tailored to Jordan, a teen who needs a simple, ADHD-friendly view of privacy controls. The layout supports fast scanning with clear hierarchy, chunked content cards, and action-focused messaging (what changed, what needs review, and what’s getting safer).

My Role

  • Designed the mobile UI and dashboard structure using an 8-pt grid approach
  • Created an ADHD-friendly visual hierarchy (chunking, reduced clutter, consistent patterns)
  • Developed core feature concepts for Dashboard, Data Management, and Settings

Key Features

  • Data Snapshot Dashboard: a calm, card-based overview of storage breakdown, permissions, and recent activity
  • Permissions Insights: shows what’s public, shared with adults, or private, with clear next steps
  • Data Summaries: quick metrics like exposure progress, app permissions review, trackers blocked, and security alerts
  • Data Management: organize, filter, compare categories, and manage streams/import sources
  • Settings & Controls: export/download tools, app-level privacy preferences, and security toggles

Design Priorities

  • Clarity first: “what matters right now” is always visible
  • Low cognitive load: chunked cards, consistent spacing, predictable controls
  • Actionable guidance: gentle warnings (“review needed”) instead of alarmist language
  • Trust & calm: clean layout and readable typography to support confident decisions
