Discover Downtown Washington

A UX project centered on revitalizing the digital presence of Washington, Indiana. My team designed a centralized and user-friendly website to highlight the city's rich cultural heritage, community events, and local attractions.

Project Overview

This project explored the unique cultural and historical identity of Washington, Indiana. Through user interviews, research, and testing, we built a site that serves residents, tourists, and businesses by promoting key destinations, historical insights, and upcoming events.

My Role

  • Conducted stakeholder interviews with local community leaders
  • Helped design site structure, navigation, and homepage layout
  • Created interactive wireframes and collaborated on usability testing

Key Features

  • Interactive map of local businesses and attractions
  • Dynamic calendar of city events and festivals
  • Mobile-responsive design for accessibility on the go

Methods Used

  • Low-Fidelity to High-Fidelity Prototypes: Developed initial sketches and wireframes, then translated designs into a full interactive prototype using Wix.com.
  • Wizard of Oz Method: Simulated key website features (like settings and web-page flow) during user tests to gauge interaction before backend implementation.
  • Critique Protocols: Participated in peer critiques and team feedback loops to iteratively refine layout, visual hierarchy, and user flows.
  • Social Media Data Scraping: Analyzed hashtags and engagement data from Reddit to uncover popular local venues and events for content prioritization.
