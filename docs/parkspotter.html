ParkSpotter

A real-time reporting tool designed to improve visitor experiences in theme parks. Guests can report maintenance or cleanliness issues using a live, interactive map interface that notifies staff instantly.

Project Overview

ParkSpotter is a mobile-first app prototype created as part of a senior year capstone project at Indiana University. The goal was to enhance communication between theme park visitors and staff by creating an easy-to-use platform for reporting location-specific issues.

My Role

  • Conducted user research and stakeholder interviews
  • Created a SQL database for a consistent flow of user/employee inputs
  • Tested usability with target users and iterated based on feedback

Key Features

  • Live interactive park map with pin-drop functionality
  • Issue categorization (cleanliness, maintenance, safety, etc.)
  • Real-time notification to relevant park staff
